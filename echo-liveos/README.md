## 📖 **UŻYCIE**

### Podstawowe użycie

```javascript
const EchoLiveOS = require('./src/EchoLiveOS')

// Inicjalizacja systemu
const echo = new EchoLiveOS({
  quantumMode: true,
  ethicalMode: 'strict',
  creativeMode: 'enabled'
})

// Uruchomienie
await echo.startup()

// Przetwarzanie żądania
const response = await echo.processRequest({
  type: 'question',
  query: 'Jak mogę poprawić swoje życie?',
  domain: 'personal_growth'
})

console.log(response.response)
```

### Przewidywanie przyszłości

```javascript
// Przewidywanie potrzeb użytkownika
const prediction = await echo.predictUserFuture('user123', {
  currentActivity: 'shopping',
  context: 'mobile_app'
}, 'short_term')

console.log('Przewidywane potrzeby:', prediction.predictions)
```

### Generowanie twórczych rozwiązań

```javascript
// Twórcze rozwiązanie problemu
const solution = await echo.generateCreativeSolution(
  'Jak zmniejszyć zmiany klimatyczne?',
  'environmental'
)

console.log('Innowacyjne rozwiązanie:', solution)
```

### Wkład w kolektywną inteligencję

```javascript
// Anonimowy wkład w system
await echo.contributeToCollective('user123', {
  type: 'insight',
  content: 'Ludzie cenią sobie autentyczność',
  context: 'social_observation'
}, {
  timestamp: Date.now(),
  activity: 'reflection'
})
```
