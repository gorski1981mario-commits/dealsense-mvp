=== KROKI SETUP ===

🔧 KROK 1: Google Cloud Console (5 minut)
1. Wejdź na: https://console.cloud.google.com/
2. Zaloguj się swoim Google account
3. Stwórz nowy projekt: "DealSense Shopping"
4. Włącz API:
   - Custom Search API
   - Google Shopping Content API

🔧 KROK 2: Custom Search Engine (5 minut)
1. Wejdź na: https://cse.google.com/
2. Stwórz nową Search Engine:
   - Sites to search: "www.google.com/shopping"
   - Language: "Dutch"
   - Search entire web
3. Skopiuj Search Engine ID

🔧 KROK 3: Klucze API (3 minuty)
1. W Google Cloud Console → Credentials
2. Stwórz API Key
3. Ogranicz klucz do swoich API
4. Skopiuj klucz

🔧 KROK 4: Dodaj do .env (2 minuty)
W server/.env dodaj:
GOOGLE_SHOPPING_PAID_API_KEY=twój_klucz_api
GOOGLE_SEARCH_ENGINE_ID=twoje_search_engine_id
