❌ Brak uprawnień kamery

=== ROZWIĄZANIE ===

🔧 KROK 1: Sprawdź uprawnienia w app.json
{
  "expo": {
    "plugins": [
      [
        "expo-camera",
        {
          "cameraPermission": "Allow DealSense to access your camera to scan product barcodes."
        }
      ]
    ]
  }
}

🔧 KROK 2: Dodaj requestPermissions w kodzie
import { Camera } from 'expo-camera';

const requestCameraPermission = async () => {
  const { status } = await Camera.requestCameraPermissionsAsync();
  if (status === 'granted') {
    console.log('Camera permission granted');
  } else {
    console.log('Camera permission denied');
  }
};
