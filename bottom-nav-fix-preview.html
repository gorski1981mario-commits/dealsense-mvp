🎯 Proponowane Zmiany CSS

✅ REKOMENDACJA (idealnie wycentrowane)

padding: 0 20px; /* tylko boki */
display: flex; /* zamiast grid */
align-items: center; justify-content: center;
.nav-item { flex: 1; height: 44px; gap: 3px; padding: 8px 4px; }

Dlaczego: Flex + align-items center = idealne centrowanie w pionie. Height 44px dla przycisków = równe marginesy góra/dół (8px każdy)

Opcja 2: Jeszcze mniejszy padding

padding: 6px 20px;
.nav-item { gap: 1px; padding: 4px 2px; }

Efekt: Więcej miejsca, ale może wyglądać za ciasno

Opcja 3: Bez gap w nav-item

padding: 10px 20px;
.nav-item { gap: 0px; padding: 5px 4px; }

Efekt: Ikona i tekst bliżej siebie
